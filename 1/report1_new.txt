

**审查情况总述**：本次审查涵盖了169条SNL语句和35条条文，其中有效的SNL占比14.8%，有效条文占比42.9%。审查中发现，有效SNL的通过率为48%，有效条文的通过率为40%，表明存在较多不符合规范的情况，需进一步整改。

**分类展示审查结果**：

**1. 疏散门宽度不足**：
- **错误**：疏散门宽度为831.0mm，小于规范要求的900mm。
  >> 构件名称：门-单扇门-M_10 疏散门
  >> 构件位置：中后(x=18332.5, y=31608.5, z=0.0)
  >> 涉及条文名称：GB 50016-2014《建筑设计防火规范》（2018年版）.spl5.5.18
  >> 原文选摘：疏散门和安全出口的净宽度不应小于0.9m。

- **错误**：疏散门宽度为1230.0mm，小于人员密集场所要求的1400mm。
  >> 构件名称：门-双扇门-M1222 外门 疏散门 安全出口 疏散外门
  >> 构件位置：中心(x=18367.5, y=14562.5, z=0.0)
  >> 涉及条文名称：GB 50016-2014《建筑设计防火规范》（2018年版）.spl5.5.19
  >> 原文选摘：人员密集场所的疏散门净宽度不应小于1.40m。

**2. 自动灭火装置缺失**：
- **错误**：建筑面积大于1000m²的餐馆未设置自动灭火装置。
  >> 构件名称：无
  >> 构件位置：无
  >> 涉及条文名称：GB 50016-2014《建筑设计防火规范》（2018年版）.spl8.3.11
  >> 原文选摘：餐馆烹饪操作间应设置自动灭火装置。

**3. 排烟系统问题**：
- **错误**：标高1的疏散走道长度超过20m，未设置排烟设施。
  >> 构件名称：标高-标高 1
  >> 构件位置：无定位
  >> 涉及条文名称：GB 50016-2014《建筑设计防火规范》（2018年版）.spl8.5.3
  >> 原文选摘：建筑内长度大于20m的疏散走道应设置排烟设施。

**4. 灭火器数量不足**：
- **错误**：公共娱乐餐饮场所灭火器数量不足，需增加至房间面积的0.629%。
  >> 构件名称：标高-标高 1
  >> 构件位置：无定位
  >> 涉及条文名称：GB 50140-2005《建筑灭火器配置设计规范》.spl5.2.1
  >> 原文选摘：A类火灾场所的灭火器保护距离应符合表5.2.1规定。

**5. 安全出口标志灯问题**：
- **错误**：安全出口标志灯周围2400mm内无应急照明灯。
  >> 构件名称：指示灯-安全出口标志灯-0
  >> 构件位置：中心(x=18366.5, y=15073.5, z=0.0)
  >> 涉及条文名称：GB 55037-2022《建筑防火通用规范》.spl10.1.9
  >> 原文选摘：疏散照明应设置在安全出口、疏散走道等部位。

**6. 防火门问题**：
- **错误**：公共歌舞娱乐场所无防火门。
  >> 构件名称：建筑-建筑
  >> 构件位置：无定位
  >> 涉及条文名称：GB 55037-2022《建筑防火通用规范》.spl4.3.7
  >> 原文选摘：歌舞娱乐场所房间之间应采用耐火极限不低于2.00h的防火隔墙分隔。

**7. 门级别标记不足**：
- **错误**：门未标记为甲级或乙级防火门。
  >> 构件名称：门-单扇门-M0922 疏散门
  >> 构件位置：中心(x=26249.5, y=16839.5, z=0.0)
  >> 涉及条文名称：GB 55037-2022《建筑防火通用规范》.spl6.4.3
  >> 原文选摘：特定部位的门应不低于乙级防火门，建筑高度大于100m的为甲级防火门。

**整改建议**：

1. **增加疏散设施**：在歌舞娱乐场所设置地面方向标志灯，确保疏散门宽度符合规范要求。
2. **安装自动灭火装置**：在餐馆厨房和明火操作区域安装自动灭火装置。
3. **完善排烟系统**：在疏散走道超过20m的区域设置排烟设施，并提供相关平面图或布置图。
4. **补充灭火器**：根据房间面积增加灭火器数量，确保配置符合规范要求。
5. **加强疏散照明**：在安全出口标志灯周围设置应急照明灯，确保疏散路径照明充足。
6. **设置防火门**：在公共歌舞娱乐场所的房间之间设置耐火极限不低于2.00h的防火隔墙，并确保防火门数量符合要求。
7. **标注门级别**：对疏散门和其他关键部位的门进行甲级或乙级防火门的标记，确保符合规范要求。

以上为本次审查的主要问题及整改建议，请尽快落实整改措施，确保建筑消防安全规范齐全。
